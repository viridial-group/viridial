# US-FE-05: Neighborhood Insights UI & Mini-Map (IHM)

## Status: Draft

### Story
En tant que visiteur, je veux visualiser des informations sur le quartier via un panel insights et une mini‑map pour juger de la qualité de vie autour d'une annonce.

### Acceptance Criteria
- Panel insights avec score composite, distances aux écoles/gares, liste d'aménités (500m) et données démographiques si disponibles.
- Mini‑map avec heatmap des commodités et possibilité de toggler layers.
- Provider adapter UI (empty state, provider selection) pour debug/data provenance.
- Responsive et accessible (labels, legends, colorblind‑friendly palette).

**Priority:** P0
**Estimation:** 3

### Tasks
- [ ] Wireframes panel insights + mini‑map
- [ ] Implement visualizations: score card, list, mini‑map layer controls
- [ ] Color palette for heatmap accessible (colorblind modes)
- [ ] Prototype with mock data and usability check

### Definition of Done
- [ ] Maquettes validées et components ajoutés au design system
- [ ] Prototype fonctionnel pour PO review
- [ ] Accessibility checklist passed
