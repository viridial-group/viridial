# US-022: Virtual Tours & Rich Media

## Status: Draft

### Story
En tant qu'acheteur potentiel, je veux visualiser des visites virtuelles (360° / vidéo / floorplans) pour mieux apprécier l'espace sans déplacement.

### Acceptance Criteria
- Support upload of 360 photos / MP4 tours and display via embedded viewer.
- Transcoding pipeline for acceptable formats and creation of thumbnails.
- Viewer supports hotspots and floorplan overlay.
- Storage & CDN integration; usage analytics for tours.

**Priority:** Medium (P1)
**Estimation:** 8

### Tasks
- [ ] Media ingestion & transcoding pipeline
- [ ] Viewer integration (360°/video/floorplans)
- [ ] UI: tour tab and analytics hooks
- [ ] Tests & performance budgets
